## 🎯 PHASE 2: SINGLE-CORRECT MCQ STRATEGY

> [!WARNING]
> **JEE Advanced CBT SCRAMBLES options per candidate!**
> Letter-based strategies ("Always pick A in Physics") are DEAD WRONG and will fail you in the exam.

You must rely on **Option Content Analysis** (which survives scrambling):

### 1. The Option Length Myth
* Previous advice said "the longest option is correct". **Data proves this FALSE.**
* The longest option is correct only ~16-25% of the time. 
* The **shortest** or **middle-length** options are mathematically more likely to be correct. Don't blindly pick the longest block of text.

### 2. The Absolute Statement Tell
* Options with words like "always", "never", "only", "all" → usually **WRONG** (eliminate them).
* Options with "may", "can", "sometimes", "under certain conditions" → usually **RIGHT**.

### 3. The Extreme Value Tell
* In MCQs where options are numerical values (e.g., 5, 10, 50, 100), the answer is rarely the highest or lowest value.
* It's almost always one of the middle values. Eliminate the extremes.

### 4. Dimensional Analysis
* In Physics and Physical Chemistry, check if the UNITS of the options make sense.
* Eliminate options with wrong dimensions → guess from the remaining.

**WHEN TO GUESS:**
* If you can eliminate even 1 option → **GUESS** (+0.33 Expected Value)
* If you can eliminate 2 options → **ALWAYS GUESS** (+1.0 Expected Value)

## 🎯 PHASE 3: MULTI-CORRECT MCQ STRATEGY

This is where most people lose marks. **MULTI-CORRECT QUESTIONS ARE A TRAP.**

| Finding | Data |
|:---|:---|
| Most common # of correct options | **2 options** (56% of the time) |
| Almost never all 4 | Only 1.4% |
| Attempting blindly | **Net negative expected value** (-4 marks over the exam) |

### Your Multi-Correct Protocol:

```
Step 1: Read the question. If you have ZERO CLUE:
        SKIP ENTIRELY. Do not touch it. The -2 penalty will kill your score.
        
Step 2: If you are 100% SURE one option is correct but unsure of others:
        Mark ONLY that ONE option.
        If right = +1 mark (partial credit). No penalty for missing others.
        
Step 3: NEVER blindly mark 2 or 3 options hoping for partial marks. 
        A single wrong option triggers the -2 penalty.
```
