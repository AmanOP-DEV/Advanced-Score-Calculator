## 🎯 TRICK CATEGORY 2: SINGLE-CORRECT MCQ ELIMINATION (15 Heuristics)

Pure random guess EV = 0.00. You MUST eliminate at least 1 option to make guessing profitable.
Eliminate 1 → EV = **+0.33**. Eliminate 2 → EV = **+1.00**.

### Heuristic 1: The Opposite Options Rule ⭐⭐⭐
**If two options are direct opposites (increase/decrease, positive/negative, true/false), one of them is almost certainly correct.**
- Why: Test-writers create distractors by negating the correct answer
- Action: Eliminate the other 2 options → 50/50 between the pair → EV = +1.00

### Heuristic 2: The Outlier Elimination ⭐⭐⭐
**If 3 options share a similar pattern/range and 1 is wildly different, the outlier is usually WRONG.**
- Numerical options: If three are close (12, 14, 15) and one is far (102) → eliminate 102
- Conceptual: If 3 options discuss similar physics and 1 is about something unrelated → eliminate it

### Heuristic 3: Absolute Statement Detector ⭐⭐⭐
**Options with "always", "never", "only", "all", "must" → usually WRONG.**
- These don't allow for exceptions, which science always has
- Options with "may", "can", "sometimes", "generally", "often" → usually RIGHT
- This is backed by academic MCQ research across multiple studies

### Heuristic 4: Option Length Analysis ⭐⭐
**Correct options are 1.25x longer on average (37.5 chars vs 30.0 chars).**
- From our 49-question verified analysis (2025 papers):
  - Longest option correct: **31%** of the time
  - Shortest option correct: only **20%** of the time
  - 2nd longest correct: **29%** (the sweet spot)
- **Strategy:** If you can't eliminate anything else, lean toward the **2nd longest or longest** option

### Heuristic 5: Extreme Value Elimination ⭐⭐⭐
**When options are numerical values, the highest and lowest are usually WRONG.**
- The correct answer tends to be one of the middle values
- This eliminates 2 options → EV = +1.00
- Works especially well in Physics (forces, energies, wavelengths)

### Heuristic 6: "Zero" and "Infinity" Killer ⭐⭐
**If an option says the answer is "zero" or "infinity" or "undefined" → almost always WRONG.**
- Real physics/chemistry problems rarely have trivial solutions
- Exception: If the question specifically asks "when does X become zero?"

### Heuristic 7: Dimensional Analysis (Physics/Physical Chemistry) ⭐⭐⭐
**Check if the UNITS of the options make sense.**
- Force must be in kg·m/s² (or Newtons)
- Energy in Joules (kg·m²/s²)
- If an option has wrong dimensions → eliminate instantly
- Even without solving, this can kill 1-2 options

### Heuristic 8: Boundary/Extreme Case Testing ⭐⭐
**Plug in x=0, x=1, or x→∞ into the options to eliminate impossible ones.**
- If the function should be 0 when x=0, eliminate options that give non-zero
- If the answer must be positive, eliminate negative options
- Takes 10 seconds, can eliminate 1-2 options

### Heuristic 9: The "Half-Right" Trap Detector ⭐⭐
**Beware options that are partially correct but contain a wrong second clause.**
- Test-writers create plausible distractors by mixing truth with error
- If an option has two statements joined by "and" or "because", check BOTH parts
- If either half is wrong → the whole option is wrong

### Heuristic 10: Similar Options Clustering ⭐⭐
**If 3 options look very similar with small differences, the answer is likely among them (not the outlier).**
- When the test-writer creates 3 variations around the correct answer, one IS the correct answer
- The "different-looking" 4th option is usually a throwaway distractor

### Heuristic 11: Grammar/Syntax Clue ⭐
**The correct option should be grammatically consistent with the question stem.**
- If the stem ends with "a" → answer starts with consonant
- If the stem ends with "an" → answer starts with vowel
- Less reliable in math/science but still worth checking

### Heuristic 12: The "Most Complete" Option ⭐⭐
**The option that is most inclusive or comprehensive tends to be correct.**
- If option A says "X increases" and option C says "X increases due to Y" → C is more likely
- The complete answer needs more qualification to be accurate

### Heuristic 13: Cross-Question Intelligence ⭐
**Later questions sometimes contain information that helps answer earlier ones.**
- A numerical value mentioned in Q15 might be the answer to Q7
- Scan all questions in a section before answering any

### Heuristic 14: The Matching Type Hack ⭐⭐
**In matching/matrix-match questions, each item in Column I typically matches a different item in Column II.**
- Use process of elimination: if you're sure A→P, then B≠P
- Even knowing 1-2 matches can help deduce the rest

### Heuristic 15: Paragraph-Based Information Mining ⭐⭐⭐
**The paragraph gives you FREE INFORMATION. Read it even if you can't solve.**
- Extract constants, conditions, and constraints mentioned
- These directly eliminate options that violate stated conditions
- Often the paragraph contains enough info to eliminate 1-2 options without any calculation

**Multi-correct questions have NEGATIVE expected value when guessing blindly.**
Blind attempt costs you ~4 marks across the exam vs skipping.

| # Correct Options | Frequency | Notes |
|:----------------:|:---------:|:------|
| 2 | **56%** | Most common by far |
| 3 | 42% | Second most common |
| 4 | 1.4% | Almost never |

### Paper-Specific Insight
- **Paper 2:** 81% have exactly 2 correct options (much higher than Paper 1's 50%)
- **Paper 1:** More balanced between 2 and 3 correct options

### The Only Safe Multi-Correct Strategy

```
DECISION FLOWCHART:

Q: Do I have ZERO clue about this question?
├── YES → SKIP. Do NOT touch it. (-2 penalty will kill your score)
│
└── NO → Can I identify at least 1 option I'm 100% SURE is correct?
    ├── NO → SKIP.
    │
    └── YES → Mark ONLY that 1 option.
        Result: +1 mark (guaranteed partial credit, zero risk)
        
        Q: Am I also 100% SURE about a 2nd option?
        ├── NO → STOP. Don't add uncertain options.
        └── YES → Mark that 2nd option too.
            If both are in the correct set: +2 marks
            Risk: If either is wrong: -2 marks
```

### Expected Value Table for Multi-Correct

| Scenario | EV | Verdict |
|:---------|:---:|:------:|
| Know 1 option, mark only that | **+1.00** | ✅ GOOD |

| Blind guess 1 random option | **-0.50** | ❌ BAD |
| Blind guess 2 random options | -0.33 | ❌ BAD |
| Skip entirely | 0.00 | ✅ SAFE |

### Monte Carlo Proof: Skip > Attempt
From 100,000 simulations:
- **Skip multi-correct:** Mean = 17.1 marks
- **Attempt multi-correct:** Mean = 12.9 marks
- **Difference: -4.2 marks from attempting blindly**

## 🎯 TRICK CATEGORY 4: PSYCHOLOGICAL & COGNITIVE TRICKS

### Trick 1: Read the Question BEFORE Looking at Options
- Form your own answer first → prevents distractors from biasing you
- If your answer matches an option → high confidence, mark it

### Trick 2: First Instinct is Usually Right
- Research shows changing answers is wrong 60% of the time
- Unless you find a CLEAR logical reason to change, stick with your first pick

### Trick 3: Beat Decision Fatigue
- 51 questions × 2 papers = 102 decisions
- Your brain gets worse at decisions over time
- **Do numericals LAST** (they need zero decision-making, just enter the default)
- Do your strongest subject FIRST while your brain is fresh

### Trick 4: The 30-Second Rule
- If you've been staring at a question for 30 seconds without progress → MOVE ON
- Come back in the second pass with fresh eyes
- Don't let one hard question steal time from 3 easy ones

### Trick 5: Anchor Bias Exploitation
- If the question mentions a specific number (e.g., "a force of 10N"), the answer often relates to that number
- Not the same number, but in the same order of magnitude
